# Ecommerce_Project

A fully responsive e-commerce website built using HTML, CSS, and JavaScript. This project showcases a modern online shopping experience with interactive features.

 Features:-
 Responsive Design – Works on all screen sizes.
 Navigation Bar – Includes a logo and navigation links.
 Product Gallery – Displays products with images and descriptions.
 Image Slideshow – Auto-sliding images for featured products.
 Shopping Cart – Allows users to add and remove products.
 Interactive UI – Smooth animations and transitions.
